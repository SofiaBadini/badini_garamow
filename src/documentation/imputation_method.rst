Imputation method
******************

Documentation of the code in *src.imputation_method*.

Define imputation methods
=========================

``imputation_method.py``
-------------------------

The python script ``imputation_method`` defines the imputation methods we use
to conduct the sensitivity analysis (see python script ``analysis'').

.. automodule:: src.imputation_method.imputation_method
    :members:


``test_imputation_method.py``
------------------------------

The python script ``test_imputation_method`` contains a testing suite for the
imputation methods. This ``test_imputation_method`` is run everytime
``imputation_method.py`` is modified.
